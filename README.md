# 项目说明

本项目包含用于模型训练、模型测评以及图形化用户界面（GUI）交互的完整代码结构。通过命令行或图形界面，用户可以方便地重新训练模型、对模型进行测试，以及查看模型在给定输入上的标注效果。

---

## 📁 项目结构

```
named_entity_recognition/
│
├─ ckpts/                 # 存放保存的模型文件
│   ├─ bilstm.pkl
│   ├─ ...
├─ DataNER/               # 存放用于模型训练的数据集
│   ├─ dev.char.bmes
│   ├─ ...       
├─ gui_app/               # 图形化用户界面
│   ├─ __init__           
│   ├─ main.py            # 图形化用户界面入口
│   ├─ model_workflow.py    
│   ├─ popup_window.py                                      
├─ models/                # 各类模型定义与存储目录
├─ TransData/             # 原始数据集与预处理文件
├─ utils/                 # 工具函数模块
├─ main.py                # 重新训练全部模型的入口脚本
├─ test.py                # 测评所用模型的脚本
├─ analysis.ipynb         # 模型错误预测分析
├─ output.txt             # 实验结果记录
└─ ...                    # 其他辅助脚本与文件
```

---

## 🚀 使用指南

### 1. **重新训练全部模型**

在项目根目录下运行：

```bash
python main.py
```

此脚本会执行所有模型的重新训练流程。根据模型数量和数据规模，训练过程可能需要一定时间。

---

### 2. **重新测评模型**

在项目根目录运行：

```bash
python test.py
```

该脚本会加载训练好的模型并执行评测流程。
评测的参考结果可在 `output.txt` 中查看，这是作者在实验中得到的预期输出。

---

### 3. **运行图形化用户界面（GUI）**

进入 `gui_app` 目录并运行：

```bash
python gui_app/main.py
```

成功运行后将弹出一个交互窗口，界面支持以下功能：

* **选择模型并执行训练**
* **选择模型并执行测评**
* **输入自定义测试语句，查看模型的训练标注效果**

GUI 更适合希望避免命令行操作的用户，也便于快速尝试模型效果。
